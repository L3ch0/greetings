# 🎂 Сайт-привітання з Днем народження

Інтерактивний сайт-привітання з анімованим тортом зі свічками та подарунками.

## 📋 Що це

Односторінковий сайт з трьома сценами:
1. **Торт зі свічками** — клікайте на полум'я щоб задути свічки
2. **Подарункові коробки** — відкривайте коробки щоб побачити вміст (спогади, компліменти, обіцянки, фото, голосові повідомлення, цитати)
3. **Фінальне привітання** — спеціальне повідомлення з конфеті

## 🎨 Персоналізація

Відредагуйте файл `lib/config.ts` щоб змінити:
- Ім'я та нік адресата
- Кольори та шрифти
- Кількість свічок
- Вміст подарунків
- Фінальне повідомлення

### Приклад налаштування подарунків

```typescript
gifts: [
  {
    type: 'memory',
    title: 'Наша перша зустріч',
    content: 'Текст спогаду...',
    image: '/images/photo.jpg', // опціонально
  },
  {
    type: 'compliment',
    title: 'Що я в тобі люблю',
    content: 'Текст компліменту...',
  },
  // ... інші подарунки
]
```

### Типи подарунків

- `memory` — спогад з опціональним фото
- `compliment` — комплімент
- `promise` — обіцянка з іконкою серця
- `photo` — фото з підписом
- `voice` — аудіоповідомлення (MP3)
- `quote` — вірш або цитата

## 📸 Додавання медіафайлів

1. **Фото**: помістіть у `public/images/`
   - Використовуйте в config: `image: '/images/photo.jpg'`

2. **Аудіо**: помістіть у `public/audio/`
   - Використовуйте в config: `audio: '/audio/message.mp3'`

## 🚀 Запуск локально

```bash
# Встановити залежності
npm install

# Запустити dev сервер
npm run dev
```

Відкрийте http://localhost:3000

## 📦 Деплой на Vercel

1. Створіть репозиторій на GitHub і запуште код:
```bash
git add .
git commit -m "Birthday site"
git push
```

2. Перейдіть на https://vercel.com
3. Клікніть "Import Project"
4. Виберіть ваш GitHub репозиторій
5. Клікніть "Deploy"
6. (Опціонально) В Settings → Domains налаштуйте свій піддомен

Сайт буде доступний за адресою `https://ваш-проєкт.vercel.app`

## 📝 Структура проєкту

```
├── app/
│   ├── layout.tsx       # Головний layout
│   ├── page.tsx         # Головна сторінка з машиною станів
│   └── globals.css      # Глобальні стилі
├── components/
│   ├── CakeScene.tsx    # Сцена з тортом
│   ├── GiftsScene.tsx   # Сцена з подарунками
│   ├── GiftBox.tsx      # Компонент коробки
│   ├── GiftContent.tsx  # Вміст подарунка
│   └── FinalMessage.tsx # Фінальний екран
├── lib/
│   ├── config.ts        # Конфігурація (редагуйте тут!)
│   └── types.ts         # TypeScript типи
└── public/
    ├── images/          # Фото для подарунків
    └── audio/           # Аудіо для подарунків
```

## 💡 Поради

1. **Фото**: Оптимізуйте зображення перед додаванням (рекомендовано max 1920px ширина)
2. **Аудіо**: Використовуйте MP3 формат, рекомендовано 128-192 kbps
3. **Мобільні**: Тестуйте на реальному пристрої для кращого UX
4. **Кольори**: Можете змінити у `lib/config.ts` на будь-які HEX значення

## 🐛 Відомі обмеження

- 3D-версія торта (Three.js) тимчасово замінена на 2D CSS через конфлікт версій React
- Аудіо автоплей не працює на iOS (потрібна user interaction)
- Voice подарунки потребують файлів MP3 у `public/audio/`
